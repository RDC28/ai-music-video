# Advanced Setup

## Recommended Community Plugins
- Dataview
- Tasks
- Templater
- Calendar
- Kanban
- Git

## Why These Matter
- Dataview: dashboard queries across projects, ADRs, incidents
- Tasks: unified task workflows and recurrence
- Templater: fast note generation with variables
- Calendar: daily/weekly note navigation
- Kanban: visual execution board for projects
- Git: local versioning visibility in vault

## Plugin Setup Checklist
- [ ] Install plugins above
- [ ] Set templates folder to `90_Templates`
- [ ] Set daily notes folder to `01_Dashboard/Daily`
- [ ] Enable auto-update `updated` frontmatter via Templater (optional)
