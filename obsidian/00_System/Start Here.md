# Start Here

## Vault Purpose
This is an engineering operating system for planning, execution, decisions, and team knowledge retention.

## Daily Flow
1. Open [[01_Dashboard/Engineering OS]]
2. Capture loose items in [[99_Inbox]]
3. Convert work into project tasks in [[50_Engineering/Backlog]]
4. Log decisions in [[50_Engineering/ADRs]]
5. Update daily note

## Weekly Flow
1. Run [[00_System/Weekly Review]]
2. Move stale notes to [[40_Archive]]
3. Refresh active priorities and blockers
