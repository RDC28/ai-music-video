# Conventions

## Folder System (PARA + Engineering)
- `10_Projects`: active, deadline-bound initiatives
- `20_Areas`: ongoing responsibilities
- `30_Resources`: reference material
- `40_Archive`: inactive material
- `50_Engineering`: shared engineering artifacts (ADRs, RFCs, runbooks, incidents)

## Naming
- Notes: `Title Case`
- ADRs: `ADR-XXXX - Decision Name`
- RFCs: `RFC-XXXX - Proposal Name`
- Incidents: `INC-YYYYMMDD - Summary`

## Linking Rules
- Prefer wiki links for internal references: `[[Note Name]]`
- Always link task notes back to parent project
- Every ADR should link to impacted RFCs, runbooks, and projects

## Hygiene Rules
- Inbox to zero at least weekly
- No orphan tasks: each task belongs to one project or area
- Archive completed projects within 7 days of completion
